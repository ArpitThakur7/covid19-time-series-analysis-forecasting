# 🦠 Coronavirus COVID-19 Visualization & Prediction

A comprehensive **Data Science & Machine Learning project** that analyzes, visualizes, and predicts the spread of COVID-19 using real-world datasets.

Built with a focus on **data-driven insights, predictive modeling, and interactive exploration**, this project demonstrates strong skills in **data analysis, visualization, and ML pipelines**.

---

## 🚀 Demo Preview

👉 Run locally using Docker and explore via JupyterLab  
👉 Interactive notebooks inside `/src`

---

## ✨ Key Features

- 📊 Data visualization of global COVID-19 trends  
- 📈 Time-series analysis and forecasting  
- 🤖 Machine learning-based prediction models  
- 🧠 Data preprocessing and cleaning  
- 🧪 JupyterLab for experimentation  
- 🐳 Fully Dockerized environment  

---

## 🛠️ Tech Stack

- **Language**: Python 3.9  
- **Libraries**: NumPy, Pandas, Matplotlib, Scikit-learn  
- **Tools**: Docker, Docker Compose, Poetry, JupyterLab
